# giveaway-system

[English](README.md) | 中文

一个用于发放免费物品的轻量级系统，流程类似电商：用户可注册、保存收货地址并提交预约订单；管理员可在简洁的后台中管理用户、活动与订单。项目强调轻量与清晰，适合作为可自行定制的 PHP/MySQL 基础。

## 功能
- 用户注册与登录认证。
- 收货地址管理。
- 预约提交与历史查看。
- 基础订单状态追踪。
- 管理端用户/活动/订单管理工具。

## 使用定位
本项目是小规模无料发放的入门框架，强调可读性与可改造性，不依赖重型框架。
